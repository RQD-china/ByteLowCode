# ByteLowCode
Vue 3 + TypeScript + Vite
This is a project for the Byte YouthCamp assessment of low code platform. The project uses the "Vue 3 + TypeScript + Vite" framework.

## 功能点进展

- 顶部工具栏样式
- 左侧组件列表样式及拖拽功能
- 中间画布样式及渲染功能
- 右侧属性设置样式
- 清空画布功能
- 设置画布属性
- 主题切换/昼夜模式
